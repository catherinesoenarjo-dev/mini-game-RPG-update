**Hasil Output Mini Game RPG**
1. Menerapkan implements pada musuh slime, naga, dan zombie dan instanceOf pada arenaPertarungan
2. Tambah aksi baru untuk musuh yang bisaTerbang
3. Tambah Status TEWAS jika musuh hp nya 0
4. Tambah menjatuhkan loot jika musuh hp nya 0
